<h2>🧐 Features</h2>

Here're some of the project's best features:

*   Service Monitoring
*   Tracks the real-time status of various services (APIs)
*   Organizes services into categories/groups for better management
*   Provides live status updates for each individual service
*   Enables reporting and tracking of service-related incidents
*   Supports multiple incident states: Investigating Identified Monitoring Resolved
*   Categorizes incidents by impact level: Critical Major Minor
*   Tracks which services are affected during each incident
*   Displays a public-facing status page for end users
*   Offers bilingual support: English and Turkish
*   Shows live system status and any active incidents
